# Kenos Logo

Canonical app icon master: `kenos-icon-1024.png` (knitted K + orbit).

Derived into:

- Web / PWA: `apps/aios/static/` via `python3 scripts/generate-life-os-brand-icons.py --app aios`
- iOS AppIcon: `clients/apple/Apps/iOS/Assets.xcassets/AppIcon.appiconset/`
- macOS AppIcon: `clients/apple/Apps/macOS/Assets.xcassets/AppIcon.appiconset/`
- watchOS AppIcon: `clients/apple/Apps/watchOS/Assets.xcassets/AppIcon.appiconset/`

Replace the master and re-run the generate script + Apple size export when the logo changes.
